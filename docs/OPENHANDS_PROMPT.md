# OpenHands Prompt
Bu proje SanalParsel adlı AI destekli drone emlak video platformudur.
Stack: Next.js 14, TypeScript, CesiumJS, Supabase, ElevenLabs, OpenRouter, iyzico, FFmpeg.
Lütfen projeyi kur, npm install çalıştır, npm run build hatalarını düzelt, Cesium asset/CSS problemlerini çöz, GitHub'a push et ve Vercel deploy yap. Env değişkenleri için kullanıcıyı yönlendir: NEXT_PUBLIC_SUPABASE_URL, NEXT_PUBLIC_SUPABASE_ANON_KEY, OPENROUTER_API_KEY, ELEVENLABS_API_KEY, IYZIPAY_API_KEY, IYZIPAY_SECRET_KEY, NEXT_PUBLIC_SITE_URL.
Not: FFmpeg endpoint Vercel limitlerine takılırsa Railway veya Render backend'e taşı.
