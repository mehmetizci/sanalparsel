# SanalParsel - Premium Cinematic Real Estate Video Platform UI Design

Figma kalitesinde, mobil öncelikli premium UI tasarımı. Drone hissi veren sinematik GIS arayüzü.

## 🎯 Özellikler

### Tasarım Özellikleri
- **Mobile-First**: 430px max-width, mobil ekranlar için optimize
- **Glassmorphism**: Blur paneller, glass efektleri
- **Dark Premium Theme**: Koyu tema, #ef4444 kırmızı aksanlar
- **Drone-Feel UI**: Sinematik, uydu görüntüsü odaklı, premium GIS hissi
- **Apple Maps + Tesla UI**: Modern, minimalist tasarım dili

### Ekranlar
1. **Landing Page** - Hero, özellikler, map preview mock
2. **Login/Register** - Auth formları
3. **Dashboard** - Projeler, istatistikler, kredi takibi
4. **Project Wizard** - 5 adımlı proje oluşturma
5. **GeoJSON Upload** - Dosya yükleme arayüzü
6. **Drone Settings** - Uçuş parametreleri
7. **AI Voice Selection** - Ses seçimi, metin girişi
8. **Video Preview** - Render önizleme, branding seçenekleri
9. **Subscription/Credits** - Kredi paketleri
10. **Profile Settings** - Kullanıcı ayarları

### UI Bileşenleri
- Blur glass paneller
- Floating controls
- Büyük harita preview
- Canlı drone path animasyonları
- Kırmızı parsel outline
- Cinematic gradientler
- Progress ring'ler
- Toggle switches
